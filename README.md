# apiproxy

> A Vue.js project

## 安装构建

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 目录结构

```
├── build                      # 构建相关
├── mock                       # mock 模拟数据
├── src                        # 源代码
│   ├── api                    # 请求后台js
│   ├── assets                 # 图片等
│   ├── components             # 全局公用组件
│   ├── icons             	   # svg图标
│   ├── layout                 # layout
│   ├── router                 # 路由
│   ├── store                  # store
│   ├── styles                 # 全局样式
│   ├── utils                  # 公用方法
│   ├── views                  # 页面
│   ├── App.vue                # 入口页面
│   ├── main.js                # 入口文件
│   └── permission.js          # 权限管理
├── static                     # 静态资源文件
└── package.json               # package.json
```
